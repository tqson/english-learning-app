Giới thiệu về Git

Git là một hệ thống quản lý phiên bản phân tán (Distributed Version Control System) cực kỳ phổ biến và mạnh mẽ.

1. Đặc điểm nổi bật

  • Theo dõi sự thay đổi của code theo thời gian
  • Làm việc song song nhiều nhánh (branch)
  • Hỗ trợ làm việc nhóm hiệu quả
  • Lưu trữ lịch sử thay đổi đầy đủ

2. Các khái niệm cơ bản

  • Repository (kho chứa code)
  • Commit (lưu thay đổi)
  • Branch (nhánh phát triển)
  • Merge (gộp code)
  • Push/Pull (đẩy/kéo code)

3. Các lệnh Git cơ bản

git init
git clone <repository-url>
git status
git add .
git commit -m "message"
git push origin main

4. Lợi ích khi sử dụng Git

  • Dễ dàng quay lại phiên bản cũ
  • Làm việc nhóm hiệu quả
  • Theo dõi được ai thay đổi gì
  • Bảo vệ code khỏi mất mát
  • Tích hợp tốt với các công cụ CI/CD

5. Nền tảng phổ biến

  • GitHub
  • GitLab
  • Bitbucket
