JavaScript Array Methods: forEach, map, filter

1. forEach Method

forEach là phương thức duyệt qua từng phần tử của mảng và thực thi một hàm callback cho mỗi phần tử.

Đặc điểm:
  • Không trả về giá trị (undefined)
  • Không thể dừng vòng lặp (break)
  • Thay đổi được mảng gốc
const numbers = [1, 2, 3, 4, 5]; numbers.forEach((item, index, array) => { console.log(`Phần tử ${item} tại vị trí ${index}`); });

2. map Method

map tạo một mảng mới bằng cách thực thi một hàm transform trên mỗi phần tử của mảng gốc.

Đặc điểm:
  • Trả về mảng mới có cùng độ dài
  • Không thay đổi mảng gốc
  • Thường dùng để biến đổi dữ liệu
const numbers = [1, 2, 3, 4, 5]; const doubled = numbers.map(num => num * 2); console.log(doubled); // [2, 4, 6, 8, 10]

3. filter Method

filter tạo mảng mới chứa các phần tử thỏa mãn điều kiện được định nghĩa trong callback function.

Đặc điểm:
  • Trả về mảng mới chứa các phần tử thỏa điều kiện
  • Không thay đổi mảng gốc
  • Callback function phải trả về boolean
const numbers = [1, 2, 3, 4, 5]; const evenNumbers = numbers.filter(num => num % 2 === 0); console.log(evenNumbers); // [2, 4]

Ví dụ thực tế

const users = [ { id: 1, name: 'John', age: 25 }, { id: 2, name: 'Jane', age: 30 }, { id: 3, name: 'Bob', age: 20 } ]; // forEach users.forEach(user => { console.log(`User ${user.name} is ${user.age} years old`); }); // map const userNames = users.map(user => user.name); console.log(userNames); // ['John', 'Jane', 'Bob'] // filter const adults = users.filter(user => user.age >= 25); console.log(adults); // [{ id: 1, name: 'John', age: 25 }, { id: 2, name: 'Jane', age: 30 }]
